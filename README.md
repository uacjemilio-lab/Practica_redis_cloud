# Práctica Redis Cloud
Proyecto ejecutado en GitHub Codespaces usando Redis Cloud.
## Configuración
- Crear Secret llamado REDIS_URL
- Crear Codespace
- Abrir el repositorio desde el Codespace en Visual Code
- Instalar en la terminal las dependencias de ser necesarias mediante
pip install -r requirements.txt
## Ejecutar
- Desde la terminal ejecutar el siguiente comando:
python src/interfaz_texto
